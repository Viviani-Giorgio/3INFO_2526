Localhost

In computer networking, localhost è un hostname che fa riferimento al computer attualmente in uso per accedervi.

Loopback

Il meccanismo di loopback locale può essere utilizzato per eseguire un servizio di rete su un host senza richiedere un'interfaccia di rete fisica.

Indirizzi IPv4

Gli standard di rete IPv4 riservano l'intero blocco di indirizzi 127.0.0.0/8 per scopi di loopback.

Name Resolution

La risoluzione del nome localhost in uno o più indirizzi IP è normalmente configurata dalle seguenti righe nel file hosts.

IETF Standards

Il nome localhost è riservato per scopi di loopback dallo standard RFC 6761.

Packet Processing

L'elaborazione di qualsiasi pacchetto inviato a un indirizzo loopback è implementata nel livello di collegamento dello stack TCP/IP.

Le versioni del database MySQL differenziano l'uso dell'hostname localhost dall'uso degli indirizzi 127.0.0.1 e ::1.
